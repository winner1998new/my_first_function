First example

  • myFirstProp

    Type: number

    Purpose: store my personal data

  • anotherSecondProperty

    Type: string

    Purpose: store user comments

  • myMethod

    Type of parametres:

    propF: string

    propB: boolean

    Returned type: void/string

    Purpose: return propF if propB is true

  • anotherMethod

    Type of parametres:

    num: number

    Returned type: number

    Purpose: return double num

  • #somePrivateProp

    Type: object

    Inside properties:

    someProp: class B

    prima: number

    isImportant: boolean

    Purpose: store unimportant things
